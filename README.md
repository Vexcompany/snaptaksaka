# VDown — Premium All-in-One Media Downloader

## Overview
A premium, iOS-style all-in-one media downloader web application. Supports 12 platforms with a beautiful frosted glass UI, iOS animations, action sheets, and toast notifications.

## Supported Platforms
- TikTok (video + audio + photo slides)
- Instagram (photos & videos)
- YouTube (MP4 video with quality selector + MP3 audio)
- Facebook (HD & SD video)
- Pinterest (images & video)
- Threads (images & video)
- CapCut (template video)
- CocoFun (video, no watermark)
- SnackVideo (video)
- Spotify (audio track)
- Google Drive (any file)

## File Structure
```
├── index.js          — Express server with /api/download and /api/status routes
├── api/config.js     — API key, base URL, endpoints, and test URLs
├── package.json      — Node.js dependencies
├── vercel.json       — Vercel deployment config
└── public/
    ├── index.html    — Main downloader page (iOS-style UI)
    ├── status.html   — API status monitoring page
    └── style.css     — All styles (iOS design system)
```

## Running
- Server: `node index.js` on port 5000
- API powered by VtechAPI (api.vtech.biz.id)

## User Preferences
- iOS-style theme with glassmorphism design
- No browser-default UI elements anywhere
- Fully responsive for mobile and desktop
- Premium one-page layout
